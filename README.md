**Superstore Sales Dashboard (Tableau)**


**Overview**

This project presents an interactive Tableau dashboard to analyze sales performance across different time periods.

**Objective**

To help stakeholders understand business performance and compare key metrics such as Sales, Profit, Quantity, and Discount.

**Features**

- KPI comparison (current vs previous period)
- Dynamic filters and parameters
- Interactive dashboard design
- Clean and modern UI

**Tools**

- Tableau Public

**Key Insights**

The dashboard highlights how changes in discount strategy and product demand influence overall sales and profitability.
